Conhecimentos

HTML

Linguagem de marcação usada para estruturar e organizar o conteúdo de páginas web. Define elementos como títulos, parágrafos, links, imagens, e listas. Serve como a "estrutura" ou "esqueleto" de um site.

CSS

Linguagem usada para descrever a apresentação visual de um documento HTML. Controla o layout, cores, fontes, espaçamento e outros aspectos estéticos da página. Permite criar designs responsivos e adaptar o site para diferentes dispositivos.

JAVASCRIPT

Linguagem de programação que adiciona interatividade e dinamismo às páginas web. Usado para criar funcionalidades como animações, validação de formulários, e atualização de conteúdo sem recarregar a página (AJAX).

REACT

Biblioteca JavaScript para construir interfaces de usuário, especialmente útil para criar componentes reutilizáveis e atualizações rápidas em aplicativos de página única.

NEXT

Framework baseado em React que facilita a criação de aplicativos web otimizados e com renderização do lado do servidor. Ele é ideal para construir aplicações modernas, permitindo desenvolvimento rápido com recursos como rotas automáticas e otimização de desempenho.

FIREBASE

Plataforma de desenvolvimento de aplicativos móveis e web que fornece uma variedade de serviços, incluindo banco de dados em tempo real, autenticação de usuários e armazenamento em nuvem. Ele é amplamente utilizado por desenvolvedores para construir aplicações escaláveis e de alto desempenho.

C#

Linguagem de programação criada pela Microsoft. Ela é usada para desenvolver programas para Windows, jogos, sites e aplicativos. Muito utilizada com a plataforma .NET, é fácil de entender e poderosa para criar sistemas completos. Também é usada na criação de jogos com o Unity.
